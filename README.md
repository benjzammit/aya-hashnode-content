# AYA Hashnode Content

This repository is the dedicated Hashnode publishing source for Ask Your Audience.

## Publishing Rules

- Hashnode publishes markdown files that live at the repository root.
- The root-level `.md` files in this repo are the publishable blog posts.
- The `hashnode-drafts/` folder is editorial working material and should not be treated as the live publishing surface.

## Workflow

- Draft and optimize ideas in `hashnode-drafts/`.
- Promote finished posts to the repository root with valid Hashnode front matter.
- Keep the website application repo separate from this content repo so Vercel production deploys are not affected by blog-only content updates.
